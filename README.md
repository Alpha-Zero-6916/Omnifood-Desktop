## What's Inside

- 🍕 **Browse Menu** — Explore food categories and items with a clean, visual layout
- 🛒 **Order Interface** — Intuitive and user-friendly ordering experience
- 📱 **Fully Responsive** — Looks great on mobile, tablet, and desktop
- 🎨 **Modern UI** — Warm, appetizing design with smooth visual styling
- ⚡ **Lightweight** — Pure HTML & CSS, loads instantly with zero overhead

---

## Getting Started

```bash
# 1. Clone
git clone https://github.com/Alpha-Zero-6916/Omnifood-Desktop.git

# 2. Open
open index.html
```

No installs. No build step. Just open and go.

## Project Structure

```
Omnifood-Desktop/
├── index.html          ← Main entry point
├── css/
│   ├── general.css     ← Base & reset styles
│   ├── style.css       ← Main stylesheet
│   └── query.css       ← Responsive media queries
└── img/
    ├── meals/          ← Meal images
    ├── gallery/        ← Food gallery
    ├── customers/      ← Testimonial photos
    ├── logos/          ← Partner logos
    └── hero.png        ← Hero graphic
```
