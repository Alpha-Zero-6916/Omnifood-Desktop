🍽️ Omnifood — Food Ordering App
A modern, fully responsive food ordering website built with pure HTML and CSS. Browse menus, explore categories, and place orders — all through a clean and intuitive interface.

✨ Features

🍕 Browse Menu — Explore a variety of food categories and items
🛒 Order Interface — Clean and user-friendly ordering layout
📱 Fully Responsive — Works seamlessly on mobile, tablet, and desktop
🎨 Modern UI — Attractive design with smooth visual styling
⚡ Lightweight — Built with pure HTML & CSS, no frameworks needed
